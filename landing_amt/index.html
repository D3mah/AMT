Тема Wordpress - AMTHEME 2

Подойдет для администраторов образовательных сайтов и начинающим веб-разработчикам
Позволяет создать сайт соответствующий требованиям действующего законодательства
Поддерживает все современные браузеры

Функционал

Wordpress 5.0+ & GetUikit 3

Тема разрабатывалась под версию Wordpress 5.0 и выше. В верстке использован GetUikit 3 — это новейшая версия фреймворка, которая является наиболее популярной платформой HTML, CSS и JavaScript для разработки адаптивных сайтов. Рекомендуемая версия PHP 7.0 и выше, это обусловлено системными требованиями Wordpress. Не забывайте, что одним из требований к сайтам образовательных организации является расположение сервера на территории РФ.

Версия сайта для слабовидящих

Соответствует стандартам WCAG 2.0 (W3 Web Content Accessibility Guidelines), WAI-ARIA (Web Accessibility Initiative — Accessible Rich Internet Applications), ГОСТ Р 52872-2019. Вывод альтернативного описания при отключении изображений (берутся из атрибута alt – не забываем заполнять). Увеличение шрифта от 100% до 200%. Регулировка отступов между строками и кернинга (отступов между буквами). Шрифт с засечками и без засечек. Режим черно-белых изображений. Четыре цветовых схемы.

Функционал темы

Имеет версию для слабовидящих, навигацию (хлебные крошки), поиск по сайту. Имеется удобный "Архив Записей", записи можно отфильтровать. Раздел "Фотогалерея", в котором можно разместить фотографии с мероприятий и.т.д. Дополнительные разделы, которые можно вывести на главную страницу сайта (Например: Анонсы, Приказы и.т.д) Есть встроенный генератор карты сайта и возможность вывода страницы с html версией карты сайта. Имеется возможность подключить форму "Обращения граждан", необходимый раздел, который должен располагаться на сайте согласно Федеральному закону №355 от 27 ноября 2017 года. Структура сайта должна отвечать методическим рекомендациям описанным в Приказе Рособрнадзоре от 14.08.2020 (изм. от 07.05.2021) №831

Рекомендуемые плагины

Wordpress занимает первое место по популярности среди бесплатных CMS. В официальном репозитории вы можете скачать огромное количество плагинов (расширений). Рекомендуемый список плагинов для нашей темы приведен ниже.

Duplicator

Это многофункциональный инструмент, который позволяет вам перемещать, переносить и клонировать ваш сайт — или вы можете просто скопировать его, чтобы создать резервную копию. Более того, Duplicator является гибким, простым в использовании и очень надежным.

WP Translitera

Плагин преобразует кириллические символы в латинские в ярлыках (URL-ссылках) записей, страниц, терминов, а также в именах файлов вложений. Пригодится для создания удобочитаемых URL-адресов.

Smart Slider 3

Бесплатный адаптивный слайдер изображений и видео, который отлично работает на всех основных устройствах

TablePress

Плагин для создания красивых и многофункциональных таблиц на страницах вашего сайта, без необходимости написания кода.

Button visually impaired

Button visually impaired — это плагин, который автоматически изменяет версию вашего сайта для слабовидящих людей. Панель на сайте для слабовидящих дает возможность изменять цветовую гамму сайта, размеры шрифтов, синтезатор речи озвучит вслух изменения настроек. С помощью неё можно изменять функции сайта, которые удовлетворят потребностями людей с ограниченными возможностями.
